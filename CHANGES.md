# История изменений Ion Pulse

Формат основан на Keep a Changelog. До первого релиза все изменения записываются
в секцию `Unreleased`, а при релизе переносятся под версию и дату.

## Unreleased

### Added

- Создан основной документ продукта и план разработки `PROJECT.md`.
- Создан словарь предметной области `CONTEXT.md`.
- Зафиксированы двуязычная модель публикаций, RBAC, редакционный workflow,
  переводы, еженедельный журнал и дайджест.

### Changed

- Проект переименован в Ion Pulse.
- Корневая рабочая папка переименована из `habr` в `ion-pulse`.
- Локальная папка backend переименована из `habr-clone-api` в `ion-pulse-api`.
- GitHub-репозиторий переименован из `habr-clone-api` в `ion-pulse-api`.

### Removed

- Удалена прежняя реализация backend для разработки новой системы с нуля.

### Pending

- Выбор стека и создание нового каркаса API.
